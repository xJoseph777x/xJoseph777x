# Hi there! 👋

I'm a young software developer passionate about backend development and building the logic that powers modern applications.

What excites me the most is seeing how an action on a screen travels through APIs, reaches the server, processes business rules, interacts with databases, and then returns meaningful results to users. Watching endpoints connect the frontend with the database is something I genuinely enjoy.

## What I Love Most

While frontend interfaces and databases are important, my favorite part of software development is **everything in between**.

I enjoy designing and implementing:

- REST APIs
- Business logic
- Service layers
- Authentication and authorization
- Data processing
- Application architecture
- Backend integrations
- Database interactions through clean abstractions

For me, the most interesting challenge is transforming a user action into a complete workflow that moves through the backend and produces the expected result.
